# X Clone

A modern and responsive user interface inspired by X (formerly Twitter), built using Tailwind CSS and modern frontend development practices.

## Overview

X Clone recreates the core visual experience of a modern social media platform, focusing on clean design, responsive layouts, and intuitive user interactions.

The project was developed to strengthen frontend engineering skills, component-based thinking, and responsive web development using industry-standard tools.

## Features

* Responsive X-inspired interface
* Modern dark-themed design
* Interactive navigation sidebar
* Post creation section
* Social feed layout
* Trending topics panel
* Mobile-friendly user experience
* Clean and maintainable code structure

## Tech Stack

* HTML5
* Tailwind CSS
* Vite

## Project Objectives

* Replicate a real-world social media interface
* Practice responsive web design
* Improve frontend architecture skills
* Learn modern styling workflows using Tailwind CSS
* Build production-quality UI components

## Installation

Clone the repository:

```bash
git clone https://github.com/Sourav-Grover/X-Clone-Full-Stack-Social-Media-Platform.git
```

Navigate to the project directory:

```bash
cd X-Clone-Full-Stack-Social-Media-Platform
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
